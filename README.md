## Run examples
#### Requirements
1. Python 3.6
2. SimPy 3.0.11
3. TensorFlow 1.12.0
4. Numpy 1.15.3
5. Pandas 0.23.4
#### Install and run
1. `git clone git@github.com:RobertLexis/CloudSimPy.git`
2. Add the path to directory cloudsimpy to system environment **PYTHONPATH**
3. `cd cloudsimpy/playground/Non_DAG/launch_scripts`
4. `python main-makespan.py`
